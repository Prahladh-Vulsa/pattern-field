On the Data

This interface records intentional actions as binary signals. A 1 indicates an action was performed with intent. A 0 indicates it was not. Missing entries are treated as 0 for frequency calculations but are distinguished visually as unrecorded observations.

No value judgment is embedded in this system. Patterns are observed, not evaluated.
